# Aplicación Planificador de Gastos para Conferencias

En el proyecto de práctica de la Aplicación Planificador de Gastos para Conferencias, construirás el front end de una aplicación que permite al usuario calcular los gastos relacionados con la organización de una conferencia en un centro de convenciones.

**La Aplicación Planificador de Gastos para Conferencias tendrá una página de inicio, una página de selección de productos y una ventana emergente de
resumen. La página de selección de productos permite a los usuarios elegir sus salas, sus complementos y comidas. Basado en esas selecciones, proporciona
precios. La ventana emergente resume los gastos en una tabla según las selecciones del usuario de la página de selección de productos.**
